¿Quién inventó el sistema de control de versión Git y por qué?
fue creado por Linus Torvalds. Originalmente, la comunidad de desarrolladores de Linux utilizaba BitKeeper, el hecho de ser un software propietario y ciertas desavenencias con su creador, Larry McVoy, llevaron a Linus Torvalds a decantarse por buscar un nuevo sistema de control de versiones.
