¿Quién inventó el sistema de control de versión Git y por qué?

Github fue desarrollado por Chris Wanstrath, P. J. Hyett, Tom Preston-Werner y Scott Chacon usando Ruby on Rails,El Hub, también conocido como concentrador, fue uno de los primeros equipos utilizados para cambiar informaciones entre computadoras de una red local.

¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018 Microsoft compró GitHub por la cantidad de 7500 millones de dólares​​, El usuario común podía notar las restricciones de Microsoft en su propio trabajo diario con el ordenador. ¿Quieres editar texto? Tienes que pasar por el aro de Word. ¿Quieres instalar Linux? Antes hay que disputar una lucha a muerte con el arranque de Windows. Y así, con un largo etcétera de ejemplos.
